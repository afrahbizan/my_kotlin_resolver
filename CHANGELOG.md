# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-25

### Added
- Initial release of the automated Kotlin Gradle Plugin version resolver.
- Implemented global `subprojects` configuration hook inside `build.gradle.kts`.
- Dynamic fallback to version `1.9.20` if `kotlin_version` is missing in the root host project.
- Complete documentation and installation architecture guides.
